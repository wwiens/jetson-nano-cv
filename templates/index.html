AI3dge

This project was inspired by another project I created that detected squirrels and then triggered a water gun to scare them away. I found that it was more effective to train the model on squirrels in the local environment rather than from images on the internet.

The result was this tool that can capture, train and detect any type of object using an NVidia Jetson Nano and a USB video camera. There are 4 main sections to the project:

  1. Collect capture images of desired objects in your own space
  2. Annotate the images with bounding boxes
  3. Train a model with the collected data
  4. Detect the objects that you trained the model on.

The main reason for incorporating all of these steps into a single project was to provide a learning experience. I thought it was interesting to have all of these steps in one single application, and by providing the source code it may help others with their Jetson Nano experience.

Each step is dependant on the previous step being completed, so you must complete the steps in the order they are presented. .

This project was designed specifically for an Nvidia Jetson Nano but the code should work fine on another platform. I have tested this with the Nano and on a Windows 11 PC.
